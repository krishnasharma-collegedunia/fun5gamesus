## 🛠️ Tech Stack

- **Pure HTML/CSS/JS** — no React, no build step
- **Single-file HTML5 games** — each game is self-contained (HTML + CSS + JS inline)
- **Static hosting** — Nginx serves directly
- **Mobile-first** — 2-col grid mobile → 6-col desktop
- **GA4 + custom events** — game_play, game_select, category_filter, etc.
