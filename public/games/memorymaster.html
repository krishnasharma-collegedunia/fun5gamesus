HOW TO PLAY ▼

Digit Span

  • Watch digits flash one-by-one
  • Type them in order using the keypad
  • Each correct answer adds a digit

Pattern Grid

  • Cells light up in a 4x4 grid
  • Tap them back in the same order
  • One extra cell added per round

Sequence

  • Watch the colours flash in sequence
  • Repeat by tapping in the same order
  • Each round gets one step longer
